# Poverty Hotspot Identification System — Telangana

A full-stack application leveraging Machine Learning to identify and predict poverty hotspots in Telangana, India. This project integrates real-world NFHS-5 data with a hybrid ML pipeline to provide actionable insights for NGOs and policymakers.

## 🚀 Key Features
- **Hybrid ML Pipeline**: Combines Unsupervised Learning (K-Means) for cluster labelling with Supervised Learning (Random Forest) for predictive classification.
- **Interactive Geospatial Visualisation**: Leaflet-based map layout identifying high-poverty regions.
- **NGO Alerting System**: Automatic email notifications via EmailJS when hotspots are identified, including specific "Poverty Drivers".
- **Real-time Prediction Sandbox**: Simulation tool to test hypothetical district data against the trained model.
- **Premium UI/UX**: Modern Dashboard with Glassmorphism, built with React, Vite, and Tailwind CSS v4.

## 🛠️ Technical Stack
- **Frontend**: React 19, Vite 8, Tailwind CSS v4, Framer Motion, Recharts, Leaflet.
- **Backend**: Flask (REST API), Scikit-learn, Pandas, NumPy.
- **Services**: Firebase (Auth), EmailJS (Notifications).

## 📊 Methodology
1. **Data Source**: National Family Health Survey-5 (NFHS-5) Telangana district-level indicators.
2. **Preprocessing**: Mapping long-form survey data to a wide-form feature matrix and deriving synthetic socio-economic proxies (Income, Unemployment, Asset Ownership).
3. **Clustering**: K-Means ($k=3$) segments districts into Low, Moderate, and High Poverty groups.
4. **Classification**: Random Forest trained on cluster labels to provide real-time hotspot prediction with confidence scores.

## ⚙️ Setup & Installation

### Backend
1. Navigate to `backend/`.
2. Create and activate a virtual environment:
   ```bash
   python -m venv venv
   .\venv\Scripts\activate
   ```
3. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
4. Run the API:
   ```bash
   python app.py
   ```

### Frontend
1. Navigate to `frontend/`.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
