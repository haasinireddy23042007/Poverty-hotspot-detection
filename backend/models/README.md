# Machine Learning Models

This directory (implicitly via `backend/ml/`) manages the persistence of the trained model bundle.

## `model.pkl`
This file is a serialized Python pickle object containing:
- **KMeans Model**: Trained on NFHS-5 socio-economic indicators to cluster districts into poverty levels.
- **Random Forest Classifier**: Trained to predict the "Hotspot" status (High Poverty cluster) with a confidence score.
- **MinMaxScaler**: Used to normalize input features before prediction.
- **Feature Metadata**: Stores column names and their relative importance for Explainable AI (XAI).

## How it's Generated
The model is generated by running the `backend/ml/model.py` script. It:
1. Loads raw survey data via `preprocess.py`.
2. Performs feature engineering and normalization.
3. Fits the K-Means ($k=3$) and Random Forest models.
4. Bundles all components into `model.pkl` for the Flask API to use.

## Regeneration
To retrain the model on new data:
1. Ensure your CSV files in `data/` are updated.
2. Navigate to `backend/`.
3. Run:
   ```bash
   python ml/model.py
   ```
The Flask API also includes a `/api/retrain` endpoint that triggers this process remotely.
